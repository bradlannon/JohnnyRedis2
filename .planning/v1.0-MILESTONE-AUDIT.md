### Notable Integration Findings

**Orphaned export:** `HeartbeatPayload` and `TOPICS.heartbeat` in `@johnnyredis/shared` defined but never published or consumed. Hub uses LWT + explicit status publishes instead. No requirement impact.

**Latent type mismatch:** `sendCommand(device, board, action, value)` requires `value` as non-optional, but `CommandPayload.value` is now optional. All current callers always pass a value — no runtime break. Will matter if future valueless direct controls are added.

**Behavioral gap:** `PiezoControl` uses action `'tone'` but `SchedulerUI` action picker only offers `['on', 'off', 'set']`. Piezo cannot be scheduled as a timed tone through the UI.
