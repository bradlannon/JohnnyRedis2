SSE Events (from server GET /events):
```
event: state:init
data: {"sensors":{"motion/nano":{"device":"motion","board":"nano","value":1,"ts":1234},...},"status":{"hub":{"board":"hub","online":true,"ts":1234},...}}

event: sensor:update
data: {"device":"motion","board":"nano","value":1,"ts":1234}

event: status:update
data: {"board":"hub","online":true,"ts":1234}
```

Command API (POST /command):
```typescript
// Request body (CommandPayload from @johnnyredis/shared):
{ device: string, board: string, action: string, value: string | number | boolean }
// Response: { ok: true } on success, { error: ZodIssue[] } on 400, { error: string } on 502
```

DashboardState types (from Plan 02-02 cache.ts):
```typescript
interface SensorState { device: string; board: string; value: number; ts: number }
interface StatusState { board: string; online: boolean; ts: number }
interface DashboardState { sensors: Record<string, SensorState>; status: Record<string, StatusState> }
```
